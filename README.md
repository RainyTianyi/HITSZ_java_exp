# AircraftWar

## 游戏简介

这是一款经典的飞机大战游戏，玩家操控英雄机躲避敌机攻击并消灭敌人，通过获取道具增强自身能力，挑战更高分数。

## 游戏规则

### 基本玩法
- **操作方式**：使用鼠标拖动控制英雄机移动
- **目标**：消灭敌机获得分数，避免被敌机击中或碰撞
- **生命值**：英雄机有10000点生命值，被击中会减少生命值，被碰撞生命值会直接归零，生命值为0时游戏结束。更高级敌机的子弹伤害更高。
- **得分机制**：击毁不同敌机获得不同分数，普通敌机10分，精英敌机20分，更高级敌机分数更高

### 道具系统
游戏中有多种道具可拾取增强能力：
- **血包（Blood）**：恢复50点生命值
- **火力增强（Fire）**：5秒内发射散射子弹
- **超级火力（FirePlus）**：10秒内发射环形子弹
- **炸弹（Bomb）**：清除屏幕上部分种类敌机和子弹
- **冰冻（Ice）**：冻结/减速部分种类敌机和敌机子弹5秒

### 敌机类型
- **普通敌机（Mob）**：无攻击能力，不掉落道具
- **精英敌机（Elite）**：可射击，概率掉落道具
- **精锐敌机（Crack）**：可射击，较高概率掉落道具
- **王牌敌机（Ace）**：可射击，必定掉落道具
- **Boss敌机**：高血量强敌，掉落多个道具

## 难度模式详解

游戏提供三种难度模式，每种模式在敌机生成、Boss出现、射击频率等方面有显著差异：

### 简单模式（Simple）
- **敌机数量**：最多4架敌机同时存在
- **生成速度**：敌机生成周期较长（25个时间单位）
- **敌机分布**：60%普通敌机，20%精英敌机，15%精锐敌机，5%王牌敌机
- **射击频率**：敌机射击周期长（30个时间单位），攻击频率低
- **Boss机制**：**不生成Boss敌机**
- **难度递增**：无动态难度调整
- **适合人群**：新手玩家，熟悉游戏操作

### 普通模式（Normal）
- **敌机数量**：最多5架敌机同时存在
- **生成速度**：敌机生成周期适中（20个时间单位）
- **敌机分布**：40%普通敌机，30%精英敌机，20%精锐敌机，10%王牌敌机
- **射击频率**：敌机射击周期适中（20个时间单位）
- **Boss机制**：每获得600分生成一个Boss，Boss血量固定
- **难度递增**：每次生成Boss后，敌机速度和血量提升，但射击频率不变
- **适合人群**：有一定经验的玩家

### 困难模式（Hard）
- **敌机数量**：最多7架敌机同时存在
- **生成速度**：敌机生成周期短（15个时间单位），敌机出现频繁
- **敌机分布**：25%普通敌机，35%精英敌机，25%精锐敌机，15%王牌敌机
- **射击频率**：敌机射击周期短（12个时间单位），攻击密集
- **Boss机制**：每获得500分生成一个Boss，Boss血量随难度系数增加
- **难度递增**：
  - 每次生成Boss后，难度系数提升0.5（最高3.0）
  - 敌机速度、血量随难度系数增加
  - **射击频率动态提升**：英雄机和敌机射击周期随难度系数减小（最快可达基准的1/3）
- **适合人群**：高手玩家，追求极限挑战

## 技术特点

- **设计模式应用**：工厂模式、单例模式、观察者模式、策略模式、模板方法模式
- **面向对象设计**：良好的继承体系和接口抽象
- **动态难度系统**：基于游戏进度的自适应难度调整
- **音效系统**：背景音乐和音效支持
- **排行榜系统**：记录各难度下的玩家成绩

## 运行方式

1. 确保已安装Java运行环境
2. 编译并运行 `src/edu/hitsz/application/Main.java`
3. 在开始菜单选择难度模式开始游戏
